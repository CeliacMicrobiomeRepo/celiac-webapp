**About** 

This is the R Shiny web application used to explore the **Celiac Microbiome Repository (CMR)**. The CMR is a comprehensive collection of all available gut microbiome sequencing datasets that include individuals with celiac disease. This collection enables the research community to identify global microbial trends across diverse populations and undergo machine learning ventures that leverage all available data. 

The repository itself exists as a [GitHub repository](https://github.com/CeliacMicrobiomeRepo/celiac-repository) which contains the associated scripts, data, and documentation to facilitate future work in the field.
